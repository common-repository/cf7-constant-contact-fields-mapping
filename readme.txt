Mapping contact form 7 fields with constant contact fields 

== Description ==
Basically by default contact form 7 provide minimal [Constant Contact Integration](https://contactform7.com/constant-contact-integration/) , after integration you need to map form fields with constant contact fields as given instrcution in above integration guide then you can submit form field data to default constant contact field only.

#### How this plugin work?

This plugin enhanced fields mapping functionality so you can map any form field (tag) with any default constant contact field. After activation plugin will remove field name mandatory for mapping and you can give field (tag) name as your choise then map it with any default constant contact field.

####  How to map form field (tag) with constant contact custom field 
* You need to [buy pro version](https://checkout.freemius.com/mode/dialog/plugin/5791/plan/9460/) of plugin, in the pro version you can map any form field (tag) with any constact contact custom field.


#### How to use this plugin?  

= In WordPress Admin = 
 
Step 1: Go to Contact -> Integration -> Connecting to the Constant Contact API (follow integration guide)

Step 2: Create or Edit the Contact Form 7 form from which you want to map form field data with constant contact field. Set up the form in the Form and Mail tabs and hit "Save". Then, go to the new "Constant Contact Fields Mapping" tab.  

Step 3: On the "Constant Contact Fields Mapping" tab,  select fields which you want to map select form field (tag) from form field dropdwon then select constact contact field from constant contact dropdown and hit "Save".  Now form field (tag) are mapped wtih constact contact field which you are selected in from specific dropdown.





== Installation ==
= Automatic installation =

To do an automatic install of Contact Form 7 Constant Contact Fields Mapping, log in to your WordPress dashboard with admin credentials, navigate to the Plugins menu and click Add New.

In the search text type "Contact Form 7 Constant Contact Fields Mapping", and click Search Plugins. Once you have found this plugin please click "Install Now" button which will install plugin. Then click on "Activate" to activate this plugin.

= Manual installation =

The manual installation method involves downloading this plugin and uploading it to your web server via your favorite FTP application like filezilla. For more information please visit (https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
